# PostBot 🤖

PostBot — это интеллектуальный Telegram-бот, спроектированный на базе мультиагентной архитектуры (OpenAI GPT-5.5) для автоматического планирования, генерации и публикации контента. 

Бот работает по принципу **"Human in the Loop"**: он делает всю рутинную работу автономно, но финальное решение о публикации (апрув) всегда остается за вами.

## 🌟 Ключевые возможности

- 🧠 **Агентная Архитектура**: Три независимых агента (Planning, Generation, Chat), которые сами определяют, какие инструменты (tools) использовать для решения задачи.
- 🐙 **Интеграция с GitHub**: Бот сам анализирует ваши репозитории и README файлы, чтобы генерировать актуальный контент-план о ваших проектах.
- 👥 **Multi-Tenant (BYOK)**: Поддержка нескольких пользователей. При старте каждый автор подключает свой собственный OpenAI API ключ и GitHub PAT.
- 📸 **Поддержка медиа**: Умеет запрашивать, принимать и публиковать фотографии в канал вместе с текстом поста.
- ⏰ **Умный планировщик**: Фоновые задачи (APScheduler) следят за сроками генерации постов и отправляют эскалационные напоминания, если вы забыли согласовать пост.
- 🔒 **Защита от "галлюцинаций"**: Строгие серверные ограничения (Server-side Guards). Бот физически не может отправить пост в канал без нажатия кнопки "Опубликовать".

## 🛠 Технологический стек

- **Язык**: Python 3.11+
- **LLM**: OpenAI SDK (Function Calling / Tools)
- **Telegram**: `python-telegram-bot` (v20+)
- **База Данных**: SQLite + SQLAlchemy (ORM с поддержкой JSON-полей)
- **Фоновые задачи**: APScheduler
- **HTTP**: `requests`

## 🚀 Установка и запуск

1. **Клонируйте репозиторий и перейдите в папку проекта**:
   ```bash
   git clone <URL>
   cd PostBot
   ```

2. **Создайте и активируйте виртуальное окружение**:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте переменные окружения**:
   Скопируйте `config.py` или создайте `.env` файл (если вы используете python-dotenv) и укажите токен самого Telegram-бота:
   ```python
   TELEGRAM_BOT_TOKEN = "ваш_токен_от_BotFather"
   ```

5. **Запустите бота**:
   ```bash
   python main.py
   ```

## 📱 Как использовать (Onboarding)

1. Отправьте команду `/start` вашему боту в Telegram.
2. Бот попросит вас прислать **OpenAI API Key** (начинается на `sk-...`).
3. Далее бот попросит **GitHub Personal Access Token** для сканирования ваших проектов.
4. Наконец, добавьте бота администратором в ваш публичный или приватный Telegram-канал и отправьте боту **ID канала** (например, `@my_awesome_channel`).
5. Расскажите боту (через обычный чат) о тематике вашего канала и стиле написания. `PlanningAgent` автоматически сгенерирует для вас первый контент-план!

## 🧩 Структура проекта

- `main.py` — точка входа, настройка логирования, регистрация хендлеров Telegram.
- `models.py` — SQLAlchemy модели базы данных (`AuthorProfile`, `ContentPlan`, `PostsHistory`, `AgentState`).
- `config.py` — загрузка системных настроек.
- `database.py` — подключение к SQLite.
- `bot/handlers.py` — обработчики сообщений, коллбеков и State-Machine онбординга.
- `agents/core.py` — движок агентного цикла (Agentic Loop), управляющий вызовами инструментов (Function Calling) к OpenAI.
- `agents/instances.py` — классы конкретных агентов (ChatAgent, PlanningAgent, GenerationAgent).
- `agents/prompts.py` — системные промпты, определяющие поведение и ограничения агентов.
- `tools/schemas.py` — JSON-схемы инструментов для OpenAI API.
- `tools/executors.py` — физическая реализация инструментов (отправка постов в TG API, запросы к GitHub API).
- `scheduler/tasks.py` — фоновые задачи для запуска генерации, планирования и отправки напоминаний.
